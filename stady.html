الرئيسية قسم الطلاب عروض تواصل معنا

تعلم HTML, CSS, JS

أساسيات بناء صفحات الويب بشكل مبسط مع أمثلة عملية

HTML

هي لغة توصيف تُستخدم لبناء هيكل الصفحة. كل عنصر يُكتب داخل وسم (Tag).

مثال:

<h1>عنوان رئيسي</h1>
<!--لاحظ التعليق-->
<p>فقرة نصية</p>
<img src="image.jpg" alt="صورة">
<h2>عنوان لكن اصغر قليلا</h2>
<h3>عنوان لكن اصغر</h3>
<h4>عنوان يصغر كل ما زاد رقم</h4>

في المثال أعلاه:
- <h1> لكتابة عنوان.
- <p> لكتابة فقرة.
- <img> لإضافة صورة.
- <h2> كتابه عنوان اصغر.
- <!--coment--> لكتابه تعليق.

CSS

تُستخدم لتنسيق الصفحة: الألوان، الخطوط، الخلفيات، وتوزيع العناصر.

مثال:

body {
    background-color: black;
    font = blood;
}
h1 {
    color: blue;
    text-align: center;
}
p {
    color: white;
}
h1:hover{
    color= red;
}

في المثال أعلاه:
- الخلفية بلون الاسود.
- العنوان باللون الأزرق ومُحاذى للوسط.
- الفقرة باللون الابيض.
- عند تقريب مؤشر الفاره من العنوان يتحول لونه الي الاحمر

JavaScript

لغة برمجة تضيف التفاعل للصفحة مثل الأزرار والرسائل.

مثال:

//coment
console.log("string");
console.log(num);
let num = 0;
let str = "string";
if (num == 0){
    console.log("wow");
} else if (str == "string"){
    console.log("cool");
} else{ console.log("bad");}

في المثال أعلاه:
- في السطر الاول كيفيه كتابه تعليق.

- السطر الثاني لاحظ كيفيه طباعه نص في لوحه تحكم.

- السطر الثالث لاحظ كيفيه طباعه رقم في لوحه تحكم.

- السطر رابع اعلان عن متغير و اعطائه قيمه عدديه.

- السطر الخامس اعلان عن متغير و اعطائه قيمه نصيه.

- السطر السادس الي العاشر لاحظ كتابه جمل الشرطيه و لاحظ ان هناك علامتان يساوي.

-سطر الاخير لاحظ عدم كتابه اي شروط.
