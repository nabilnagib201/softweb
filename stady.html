تعلم HTML, CSS, JS

أساسيات بناء صفحات الويب بشكل مبسط مع أمثلة عملية

HTML

هي لغة توصيف تُستخدم لبناء هيكل الصفحة. كل عنصر يُكتب داخل وسم (Tag).

مثال:

<h1>عنوان رئيسي</h1>
<p>فقرة نصية</p>
<img src="image.jpg" alt="صورة">

في المثال أعلاه: - <h1> لكتابة عنوان. - <p> لكتابة فقرة. - <img> لإضافة صورة.

CSS

تُستخدم لتنسيق الصفحة: الألوان، الخطوط، الخلفيات، وتوزيع العناصر.

مثال:

body {
    background-color: #f8f9fa;
    font-family: Arial, sans-serif;
}
h1 {
    color: #003366;
    text-align: center;
}
p {
    color: #f39c12;
}

في المثال أعلاه: - الخلفية بلون فاتح. - العنوان باللون الأزرق ومُحاذى للوسط. - الفقرة باللون البرتقالي.

JavaScript

لغة برمجة تضيف التفاعل للصفحة مثل الأزرار والرسائل.

مثال:

<button onclick="sayHello()">اضغط هنا</button>

<script>
function sayHello() {
    alert("مرحبا بالعالم!");
}
</script>

في المثال أعلاه: - زر عند الضغط عليه يستدعي دالة. - الدالة تعرض رسالة منبثقة.
